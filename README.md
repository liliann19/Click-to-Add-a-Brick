## Description
Click to Add a Brick is a simple interactive clicker game where players build a house one brick at a time. 

Click the button to place bricks and earn points. Use those points to buy upgrades that increase how many bricks you place per click or automatically generate bricks over time. 

As you progesss, the game changes visually to reflect construction stages and reaching the final milestone completes the brick house. 

## Features 
- Click to earn bricks
- Upgrades for faster building
- Automatic brick generation
- Visual progess milestones
